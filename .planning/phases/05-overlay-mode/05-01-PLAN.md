

       - If captureRegion is null, return boundingBox as-is (full-screen capture, bbox IS screen coords unless upscaled -- check preprocessedWidth vs screen width)

       This is verified from RESEARCH.md section on OcrPreprocessor: it crops to CaptureRegion then optionally upscales 2x if height < 100px.

Use constants: FLAG_NOT_FOCUSABLE=0x00000008, FLAG_NOT_TOUCHABLE=0x00000010, FLAG_SECURE=0x00002000 as Int constants to allow unit testing without Android framework.
       - `data class OverlayTransition(val from: OverlayMode, val to: OverlayMode, val cleanupNeeded: Boolean)`
       - `object OverlayStateMachine` with `fun transition(current: OverlayMode, next: OverlayMode): OverlayTransition` -- returns transition object indicating what cleanup is needed. Same-to-same is no-op (cleanupNeeded = false).
